redis安装

redis linux install

1.下载

wget http://download.redis.io/releases/redis-4.0.14.tar.gz（版本号可以更改）

2.解压

tar xzf redis-4.0.14.tar.gz

3.转到解压后的目录

cd redis-4.0.14

4.安装

make

二进制文件被编译到src文件夹
可以转到src使用./redis-server 运行redis服务

你可以使用redis自带客户端连接redis
执行./redis-cli
输入ping，可以看到返回 “PONG”，redis已经成功安装了
